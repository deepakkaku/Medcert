# Oracle VPS + Cloudflare Deployment Guide

Follow these steps to deploy your MedCert Premium application to your Oracle VPS with industry-standard HTTPS using Cloudflare.

## 1. Oracle VPS Security List (VCN)
Before starting, ensure your Oracle Cloud VCN Ingress Rules allow traffic on:
- **Port 80** (HTTP)
- **Port 443** (HTTPS)

## 2. SSL Setup
Since you already have a wildcard certificate for `*.doctrust.in` (used by your homepage), you don't need to generate new ones.

**Ensure your current certificate files are located at:**
- `/etc/letsencrypt/live/doctrust.in/fullchain.pem`
- `/etc/letsencrypt/live/doctrust.in/privkey.pem`

If they are in a different folder, update the `ssl_certificate` paths in `medcert.conf` before uploading.

### Apply Nginx Config
1. Upload the provided `medcert.conf` to `/etc/nginx/sites-available/medcert.conf`.
2. Enable the site:
```bash
sudo ln -s /etc/nginx/sites-available/medcert.conf /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

### 5. Install dependencies and Restart PM2 on VPS
1. Run `./deploy.sh` from your local machine.
2. The script will now use the `ecosystem.config.cjs` to manage the server.

### 6. PM2 Best Practices (Run these on VPS)
To keep the app running after a reboot:
```bash
# Start the app using the ecosystem file
pm2 start ecosystem.config.cjs

# Save the process list
pm2 save

# Ensure PM2 starts on boot
pm2 startup
# (Copy and paste the command generated by the line above)
```

### 7. Log Management
```bash
# Monitor logs in real-time
pm2 logs medcert-premium

# Check process status
pm2 status
```

## 5. Directory Structure on VPS
```yaml
/home/ubuntu/medcert-premium/
  ├── dist/          # Served by Nginx
  ├── server/        # Running via PM2 (for APIs)
  ├── package.json
  └── .env
```

## Troubleshooting
- **Check Nginx Logs:** `sudo tail -f /var/log/nginx/medcert_error.log`
- **Check PM2 Status:** `sudo pm2 status`
- **Permissions:** Ensure `www-data` user can read your `/home/ubuntu/medcert-premium/dist` folder:
  `sudo chmod -R 755 /home/ubuntu/medcert-premium`
